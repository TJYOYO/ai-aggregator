# AI Aggregator

A curated collection of best practices, patterns, and workflows for AI/LLM-powered software development — aggregating knowledge across prompting, agents, tooling, and Git integration into one place.

## Why This Repo

AI tools (GitHub Copilot, Cursor, Claude, ChatGPT, etc.) are becoming core parts of the development workflow. But scattered tips and fragmented docs make it hard to build a consistent, effective practice. This repo serves as a **single aggregation point** — bringing together proven techniques from across the AI-assisted development ecosystem so you can learn, reference, and apply them quickly.

## Contents

| Directory | Description |
| --- | --- |
| [`agents/`](agents/) | **Agent instructions & AGENTS.md** — how to write project-level rules that make AI Agents behave predictably and follow your team's conventions |
| [`prompt/`](prompt/) | Prompt engineering best practices — the 5 key elements of a good task prompt, Plan vs Agent mode, session management |
| [`git/`](git/) | Git + LLM integration — auto-generating commit messages, configuring Copilot for Git workflows |
| [`mcp/`](mcp/) | MCP (Model Context Protocol) — extending agent capabilities with external tools |

## Quick Start

1. Read [`agents/Repos-Agents-instructions.md`](agents/Repos-Agents-instructions.md) to learn how to set up `AGENTS.md` for your project
2. Read [`prompt/best-practice.md`](prompt/best-practice.md) for writing effective prompts and managing AI sessions
3. Apply the patterns to your own repos and iterate
